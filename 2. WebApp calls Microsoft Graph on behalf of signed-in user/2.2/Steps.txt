1. Add the active-directory-aspnetcore-webapp-openidconnect-v2\Microsoft.Identity.Web project to the solution
1. Add Microsoft.Identity.Web as a reference in WebApp-Cache-V2 project.
1. Open up WebApp-Cache-V2 project's properties and copy the SSL Url, next to the **Enable SSL** checkbox.
1. Change the `Properties\launchSettings.json` applicationUrl to the https URl copied earlier.
1. Add a folder named `Infrastructure` and copy the contents of the Constants.cs in it
